# Titanic EDA Project – Python and AI Tra Internship

This project contains Exploratory Data Analysis (EDA) of the Titanic dataset using Python, Pandas, Matplotlib, and Seaborn.

## Folder Structure
Titanic-EDA/
   data/            → train.csv & test.csv
   notebooks/       → titanic_eda.ipynb (main analysis)
   README.md        → project overview
   
## What is done
- Loaded and inspected dataset
- Checked missing values
- Performed univariate and bivariate analysis
- Visualized key patterns
- Summarized insights with clear explanations

## Key Insights
- Females and children survived more.
- Higher-class passengers had better survival.
- Cherbourg port passengers survived the most.
- Large families had the worst survival.
- Age and Cabin columns contain significant missing values.
